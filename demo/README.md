# ggpointdensity-demo

ggpointdensity demo with [fast-kde](https://github.com/uwdata/fast-kde), [D3](https://d3js.org/), and [Vue](https://vuejs.org/).

## Development

Install [fnm](https://github.com/Schniz/fnm) (if necessary).

```bash
fnm install && fnm use && node --version
```

```bash
npm install
```

```bash
npm run dev
```
